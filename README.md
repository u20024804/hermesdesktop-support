# HermesDesktop Support

HermesDesktop is a native macOS app for running a local Hermes AI workspace.

## Support Contact

For app support, bug reports, privacy questions, or feature requests, contact:

- Email: support@clawapp.work
- GitHub Issues: https://github.com/u20024804/hermesdesktop-support/issues

Please include your macOS version, HermesDesktop app version, and a short description of the issue. Do not include API keys, Telegram bot tokens, passwords, or other secrets in public issues.

## App Store Build Notes

The App Store build runs in the macOS app sandbox and uses a restricted runtime profile. Terminal execution, browser automation, MCP/ACP integrations, scheduled background jobs, remote skill downloads, local shell execution, and non-Telegram messaging adapters are disabled. Telegram linking is optional and user configured.
